### Creating a new meet in Hytek
- Create a new folder for the new meet eg. c:\tfmeets6\PG-2026-Twi-2
- Start Hytek Meet Manager
- Open a previous meet (to use as a template)
- Use "Save As" to create a new meet file in the folder above (c:\tfmeets6\PG-2026-Twi-2\PG-2026-Twi-2)
- Use the menu "Set Up/Meet Setup" to update the name, start date and end date of the meet.
  - Update "Age Up Date" if this meet is in a different year. Normally age u date is the last day of the current year
- Use menu "File/Purge/Remove Data Selectively" and choose "Teams" to delete "Teams, Athletes, Entries/Results and Relays"
- Use the menu "Events" to edit the events for the current meet based on the events listed on trackie e.g. https://www.trackie.com/event/pearlgate-twilight-14-series-2/1037819/
- Use the menu "Events/Sessions" to create a schedule of events in order with start times
### Download Entries from Trackie
- Do one of the following options
#### Option 1: Do everything yourself
- Create an account on trackie
- Have Nick assign you as Admin for the meet
- Login to Trackie using the account created above
- Select the meet from "My Events"
  - Chose "Export Data"
    - Within the "Export Data" form
    - Make sure Hytek Meet Manager is selected
    - Choose the green "Export" button at the bottom of the form
    - This will download a Semi-colon Delimited Entries text file file to your Downloads folder
     - The file will be named after the meet followed by the date and time it was downloaded.
      -  <img width="1366" height="768" alt="Trackie-Export-Data" src="https://github.com/user-attachments/assets/a9968ecc-3614-4fe6-80c6-0e9215c8ebe1" />
- Go back to the Hytek meet you created above
- Choose File/Import/Semicolon Delimited Rosters/Entries
  - Check to make sure there are no exceptions. You will have to manually fix any exceptions.
- Choose Seeding and Seed All Events
#### Option 2: Have Nick Email the Comma Delimited Entries
- Go back to the Hytek meet you created above
- Choose File/Import/Semicolon Delimited Rosters/Entries
  - Check to make sure there are no exceptions. You will have to manually fix any exceptions.
- Choose Seeding and Seed All Events
- You are now ready to Run the meet
#### Option 3: Have someone send you a backup of the meet and restore it
- For Twligiht Meet #2, I have done this
  - Create a new folder for this meet e.g. C:\tfmeets6\2026-Twilight-2
- Make sure the backup file Tfmm6backup*.zip file I emailed you is available on the laptop computer
- Choose "File/Restore" and select the option "Unzip, Copy database to a selected folder, and open this new database"
  - use the folder you created above
